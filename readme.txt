Private feeds for ACM Cases.

== Description ==

Protects all feeds with a unique key for each user.

== Installation ==

1. Upload `cases-private-feeds` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
